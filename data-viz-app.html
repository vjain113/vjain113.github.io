Project

Visualizing Motor Vehicle Accidents using Streamlit webapp

  • Deep Visualization Project
  • Github Repository

This project uses pandas and Streamlit to create a webapp with interactive features used to visualize Motor vehicle accidents in New York City.

The dataset has been procured from the nyc open datasets - Motor Vehicle Collisions - Crashes

We used pandas for basic pre-processing of data and for providing data to streamlit objects and functions.

The app also uses pydeck to plot interactive 3-D maps and uses plotly express to create interactive charts and Finally Streamlit to deploy the webapp.
