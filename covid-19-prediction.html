Project

Predicting Covid-19 with chest X-Ray scans using PyTorch

  • Deep Learning Project
  • Github Repository

This project uses X-Ray chest scan images and using a pre-trained convolution neural network (resnet18) to classify the image to belonging to three categories of patients :

  1. Normal
  2. Viral Pneumonia
  3. Covid-19

The dataset has been procured from the kaggle database - COVID-19 Radiography Database

We also applied data augmentation techniques on the images as part of data preprocessing since we have less no. of images, especially for covid-19 patient scans

We achieved an accuracy of 97.8% on the test set.
